使用说明

  • 周报：研究区间为上周一至上周日，输出至 output/weekly/。
  • 日报：研究区间为昨日（当前日期的前一天），输出至 output/daily/。
  • 每次生成会得到三个文件：原始内容（.txt）、结构化报告（.json）、Word 最终版（.docx），文件名均以日期开头。
  • 生成过程中请勿关闭页面；完成后可直接在下方下载，或到对应目录查看。
  • API Key：选择模型后在上方输入。Gemini 需填写 E、S、G 三个 Key（用于并行研究）；千问只需一个。仅用于当次运行、不会保存；不填则使用 config.json 中已配置的 Key。
